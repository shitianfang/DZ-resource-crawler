# Discuz 论坛爬虫

- 用于采集 discuz 论坛的帖子信息和评论，可以爬取分页和层叠的评论  
- 爬取目标为 `帖子id` `用户id` `用户昵称` `发帖内容` `评论内容`  
- 本爬虫只采集数据，不做数据统计工作
- 这是临时赶制的粗糙项目，如果需要长期使用，你应该自己改写它 

食用用法
```
spider = DiscuzSpider()
data = spider.parse(3846582)
print(data)
```
